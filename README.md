# SPOT Desktop Application

#### **SPOT-Desktop is still under development**

* install **node.js**

* clone this repository
  ```bash
  git clone https://github.com/fdiblen/spot-desktop.git && cd spot-desktop
  ```

* get latest version of SPOT:
  ```bash
  npm run get-spot
  ```

* install dependencies:
  ```bash
  npm install
  ```

* build packages:
  ```bash
  npm run dist
  ```
